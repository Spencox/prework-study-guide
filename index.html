HTML

  • The head element contains information about the webpage.
  • the meta element is used to inform browser
  • The body element represents the visible content shown to the user.
    • the header tag is used for header and navigation
    • the main tag is used for general content
    • the footer tag is used for general copyright, contact, and sitemap data
  • Sematnic elements are descriptive elements used to place text
    • Headings - h1, h2, h3. h4, h5, h6
    • Paragraph - p
    • Unorderd List - ul
    • Ordered List - ol
    • List Elements - li
    • Section - section, used for sperated themes of site

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSS box model is an important concept
      • From inside out:
    1. Content
    2. Padding
    3. Border
    4. Margin

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: needed to make sure that everything is up to date before initiating a feature branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
